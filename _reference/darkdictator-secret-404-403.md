(Alternative triggers to consider: "directive4", "feelcrime")

brief burst of static, then the content renders line

## IMPLEMENTATION NOTES

- Secret page: trigger on keystrokes spelling "obey"
  in sequence (no input field — just raw keypress
  detection, same way the current slum+ trigger works).

- 404 and 403: same CRT styling as main site. Same
  scanlines, same phosphor glow, same font. They
  should feel like the system is talking to you,
  not like a broken website.

- On the secret page, consider a subtle difference:
  the green could be slightly different — a tiny bit
  more amber/yellow — to suggest you've accessed a
  different security layer. Very subtle. Just enough
  to feel wrong.

- The [ESC] [DISABLED] at the bottom of the secret
  page should match the footer gag from the main site.
